# projeto-colecionismo
1. O nosso projeto foi feito para promover a interação entre pessoas que possuem coleções das mais diversas.
2. Cada usuário pode ter mais de uma coleção com diversos itens cadastrados, tendo eles: foto, nome, descrição, categorias como cor, tamanho e raridade, além disso pode dar uma etiqueta de "Queridinho da coleção" ao item.
3. O usuário pode interagir com outros usuários (apenas seus amigos) e pode haver coméricio entre eles, além de amizades, seguidores e fóruns de determinado assunto.
4. O usuário é cadastrado com foto, nome de usuário, e-mail, telefone, data de nascimento e área de interesse.
